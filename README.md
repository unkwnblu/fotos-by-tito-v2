FotosByTito is a modern, high-performance photography portfolio website built with Next.js 16. It features a stunning visual design, smooth animations, and a responsive layout to showcase photographic work in the best light.

This project is built using the latest web technologies for speed and developer experience:

## ‚ú® Features

- **Dynamic Photo Grid**: A responsive masonry-style grid that adapts to any screen size.
- **Smooth Animations**:
  - Staggered entry animations for all sections.
  - Scroll-triggered reveals using Framer Motion.
  - Hover effects on images.
- **Loading Skeletons**: Enhanced perceived performance with skeleton loaders while images fetch.
- **Dark Mode**: Fully supported dark/light theme switching.
- **Interactive Components**:
  - Expandable FAQ section.
  - Mobile-responsive navigation menu.
  - Contact/Enquiry banner.

## üìÇ Project Structure

- **`app/`**: App Router pages and global layouts.
- **`components/`**: Reusable UI components (Navbar, PhotoGrid, About, FAQ, etc.).
- **`components/ui/`**: Low-level UI primitives (Skeleton, Buttons, etc.).
- **`lib/`**: Utility functions (e.g., `cn` class merger).
- **`public/`**: Static assets like images and fonts.
